# boilerplate-vanilaJS

boilerplate for Vanila JS FE development w/ webpack
works like SPA

## includes

-   babel transpile -> es5
-   polyfill (core-js)
-   webpack
    -   loaders
        -   css-loader, style-loader
    -   plugins
-   http-server

## Structure

-   src
    -   controllers
    -   models
-   -   index.html
